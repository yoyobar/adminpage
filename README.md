### 프로젝트 설명

`oAuth` 로그인 및 자체회원가입 기능이 존재하고 DB와 연동되어 작동하는 어드민 페이지 프로젝트입니다.<br>
개인 회원은 에 다양한 글과 댓글을 작성할 수 있으며, 어드민 페이지에서는 모든 글을 확인할 수 있고, 모든 사용자를 제거할 수 있습니다.<br>
어드민은 `admin@admin.com` / `1325`로 DB에 등록되어 있습니다.
